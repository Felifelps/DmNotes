# DMNotes

DMNotes é uma ferramenta projetada para ajudar Mestres de Jogo a organizar suas campanhas. Ela oferece recursos para gerenciar personagens, locais, eventos, itens, masmorras e fichas de forma intuitiva e fácil de usar.

## Recursos

- **Gerenciamento de Personagens**: Adicione, edite e exclua personagens com informações detalhadas como nome, classe, raça e história.
- **Gerenciamento de Locais**: Organize locais com descrições e histórias.
- **Rastreamento de Eventos**: Acompanhe eventos com títulos, descrições e histórias.
- **Inventário de Itens**: Gerencie itens com descrições detalhadas e histórias.
- **Gerenciamento de Masmorras**: Crie e gerencie masmorras com imagens e descrições.
- **Gerenciamento de Fichas**: Faça upload e gerencie fichas com imagens e descrições.
- **Suporte Multiplataforma**: Disponível como aplicativo para desktop e dispositivos móveis usando Flet, e como aplicação web usando React.

## Instalação

### Aplicativo para Desktop/Dispositivos Móveis

1. Clone o repositório:
   ```bash
   git clone https://github.com/your-repo/DMNotes.git
   cd DMNotes/desktop_mobile
   ```

2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

3. Execute o aplicativo:
   ```bash
   python src/main.py
   ```

4. (Opcional) Crie o executável independente:
   ```bash
   build.bat
   ```

### Aplicação Web

1. Navegue até o diretório `frontend`:
   ```bash
   cd DMNotes/frontend
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm start
   ```

4. Abra [http://localhost:3000](http://localhost:3000) no seu navegador.

## Uso

### Desktop/Dispositivos Móveis

- Navegue pelo aplicativo usando a barra de navegação inferior.
- Adicione, edite ou exclua entidades como personagens, locais, eventos, etc.
- Use o botão de ação flutuante para criar novas entradas.

### Web

- Acesse a interface web para gerenciar os dados da sua campanha.
- Use o cabeçalho e a navegação para alternar entre as seções.
